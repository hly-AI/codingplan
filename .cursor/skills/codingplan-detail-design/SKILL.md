# CodingPlan 详细设计

为 CodingPlan 工作流 Step 4 提供详细设计文档结构与模板。

## 文档结构

```markdown
# 详细设计：{需求名}

## 1. 模块设计
### 模块 A
- 职责
- 输入/输出
- 关键函数/类

### 模块 B
...

## 2. 数据结构
- 主要实体
- 字段说明
- 关系

## 3. 接口定义
### API/接口 1
- 路径/方法
- 请求参数
- 响应格式
- 错误码

### API/接口 2
...

## 4. 核心逻辑
- 流程说明
- 关键算法
- 状态流转

## 5. UI 组件（若涉及）
- 组件树
- Props/State
- 布局与样式

## 6. 异常与边界
- 错误处理
- 降级策略
```

## 设计原则

- 模块职责单一
- 接口定义完整
- 数据结构可扩展
- 与概要设计一致
